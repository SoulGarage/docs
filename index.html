SOUL — Документация проекта

О проекте

SOUL — это платформа для встреч, комьюнити и совместных активностей с элементами геймификации. После MVP планируется переход в Web3.

Текущий статус

  • Дизайн: 99%
  • Фронтенд: 99%
  • Бэкенд: 99%

Стек технологий

  • TypeScript
  • React + Redux Toolkit
  • NestJS
  • PostgreSQL + Prisma
  • Docker, CI/CD

Цель MVP

Запустить стабильную версию приложения, рассчитанную на 500 пользователей, протестировать ключевые гипотезы и собрать фидбэк.
